## ✅ After Running Setup

### 1. Install Dependencies (for each service)

```cmd
cd services\notification-service
pip install -r requirements.txt

cd ..\calendar-service
pip install -r requirements.txt

cd ..\search-service
pip install -r requirements.txt

cd ..\monitoring-service
pip install -r requirements.txt
```

### 2. Create Database Migrations

```cmd
cd services\notification-service
python manage.py migrate

cd ..\calendar-service
python manage.py migrate

cd ..\search-service
python manage.py migrate

cd ..\monitoring-service
python manage.py migrate
```

### 3. Create Superuser (Optional)

```cmd
cd services\notification-service
python manage.py createsuperuser
```

### 4. Start Development Server

```cmd
# Terminal 1: notification-service
cd services\notification-service
python manage.py runserver 0.0.0.0:8006

# Terminal 2: calendar-service
cd services\calendar-service
python manage.py runserver 0.0.0.0:8008

# Terminal 3: search-service
cd services\search-service
python manage.py runserver 0.0.0.0:8010

# Terminal 4: monitoring-service (WebSocket support)
cd services\monitoring-service
daphne -b 0.0.0.0 -p 8009 core.asgi:application
```

## 🛠️ Troubleshooting

### "python: command not found"
Use `python3` instead:
```cmd
python3 setup_django_comprehensive.py
```

### "ModuleNotFoundError" after running setup
You need to install dependencies:
```cmd
pip install -r requirements.txt
```

### "PostgreSQL connection refused"
Either:
1. Start PostgreSQL, OR
2. Remove DB_HOST environment variable to use SQLite locally

### "Port already in use"
Change the port in `manage.py runserver`:
```cmd
python manage.py runserver 0.0.0.0:8006  # Change 8006 to another port
```
