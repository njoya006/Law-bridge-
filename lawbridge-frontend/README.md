# LawBridge Frontend (scaffold)

This folder contains a fresh Next.js + TypeScript + Tailwind scaffold following the `LAWBRIDGE_FRONTEND_DEVELOPMENT_DOCUMENT.md` design.

Quick start

1. Install dependencies

```bash
cd lawbridge-frontend
npm install
```

2. Run development server

```bash
npm run dev
```

Notes
- This is a starting scaffold. Implement pages under `app/` and components under `components/` per the design doc.
